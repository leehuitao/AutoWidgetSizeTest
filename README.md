# AutoWidgetSizeTest

#### 介绍
可支持拉伸，自动根据行列排列，其他一些简单的小组件


    1. 可拉伸，可寻边，可缩放的widget（label）控件（可跟随窗口伸缩，可自定义排序的行列3*3、 5*5等，内部的label支持缩放）
    2. 鼠标画框功能（类似QQ截屏功能，截取返回后的位置信息的图片即可）
